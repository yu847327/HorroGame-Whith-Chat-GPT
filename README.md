# AI 生成的 HTML 恐怖小游戏

一个由 AI 生成并可继续迭代的 HTML 恐怖游戏项目。此仓库旨在提供最小化的起点，便于快速搭建、预览和扩展你的网页恐怖游戏体验。
**如何开始**

1. 网页链接https://yu847327.github.io/HorroGame-With-Chat-GPT即点即玩
2. download 分支v1.0的所有文件，或者尝试在console中进入v1.0文件根目录，运行python serve.py，然后在浏览器打开http://127.0.0.1:8000/index.html游玩

## 项目特点

- **AI 驱动**：所有文案、玩法创意与界面元素可通过大语言模型生成和迭代。
- **纯前端**：使用 HTML/CSS/JavaScript 构建，无需服务端依赖，适合部署到任意静态托管平台。
- **快速预览**：本地直接打开 `index.html` 即可运行；也可以使用本地静态服务器获得更接近线上环境的体验。
- **可扩展性**：支持持续添加场景、素材与脚本逻辑，便于迭代恐怖氛围和互动玩法。

## 仓库结构（示例）

> 初始仓库仅包含本 README，你可以参考以下约定来组织文件：
>
> - `index.html`：游戏入口页面。
> - `assets/`：图片、音效、字体等静态资源。
> - `scripts/`：JavaScript 逻辑（如场景切换、事件触发、音效控制）。
> - `styles/`：CSS 或预处理样式文件。
> - `docs/`：剧情文档、设计稿或迭代记录。

## 本地运行

1. 克隆仓库：`git clone <repo-url> && cd HorroGame-Whith-Chat-GPT`
2. 将生成的游戏文件（如 `index.html`、`assets/` 等）放入仓库根目录。
3. 预览方式：
   - 直接双击 `index.html` 在浏览器打开，或
   - 使用简易静态服务器（推荐），例如：
     ```bash
     # 使用 Python 3 内置服务器
     python -m http.server 8000
     # 或使用 Node.js 的 serve（需全局安装 serve）
     npx serve .
     ```
4. 在浏览器访问 `http://localhost:8000` 体验游戏。

## 使用 AI 迭代内容的建议流程

1. **剧情与脚本**：用提示词描述场景、角色、分支事件，让 AI 生成故事文本与交互脚本。
2. **界面与视觉**：让 AI 生成恐怖风格的配色、布局或 CSS 片段，直接整合到 `styles/`。
3. **音效与氛围**：请求 AI 给出需要的音效列表与触发时机，在 `scripts/` 中绑定播放逻辑。
4. **测试与反馈**：本地预览后，记录需要调整的文本、节奏或特效，再次通过 AI 迭代。

## 部署

- **构建与压缩**：如需优化加载速度，可使用构建工具（如 Vite、Parcel）对资源做压缩与指纹化。
